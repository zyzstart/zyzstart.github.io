The Structure Of CPP

此时a的地址为000000000061fe10，对应的十进制数值为6422032，c的地址与ab不同，因为b是a的引用，对应相同地址所以输出是等价的。

关于引用的特性

  • &起到标识作用

  • 声明引用时，需要对其初始化

  • 声明引用的类型必须和变量的类型一致

  • 引用声明完毕后不可再做其他变量的引用

  • 声明一个引用不是定义变量，只是相当于一个变量的别名，本身并非一种数据类型，不会占用存储单元

  • 不能建立数组的引用，因为数组是若干个元素的集合，无法建立数组的别名 int arr[2] = {4,5}; int &arr1[2] = arr;//是错误的

在C语言中要进行两个数交换就要使用指针，进行取值和取址操作
